🚗 Automobile Analysis Dashboard – Power BI
📌 Project Overview
The Automobile Analysis Dashboard is a data visualization project developed using Power BI to analyze automobile data and gain insights into different types of vehicles.
This dashboard focuses on comparing two vehicle categories: Convertible and Saloon, helping users understand differences in pricing, performance, and other automobile characteristics. The project demonstrates how data visualization can be used to analyze automobile industry data and support data-driven decision-making.

🎯 Objectives
Analyze automobile data for different vehicle types
Compare Convertible and Saloon vehicles based on key attributes
Identify trends and patterns in automobile pricing and specifications
Present insights through interactive and easy-to-understand visualizations

🛠 Tools & Technologies Used
Power BI – Dashboard development and visualization
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Creating calculated measures
Excel Dataset – Data source used for analysis

📊 Dashboard Features
Interactive filters and slicers for dynamic data exploration
Comparison between Convertible and Saloon vehicles
Visual representation of automobile price distribution
Insights into vehicle specifications and performance
User-friendly and interactive dashboard layout

📈 Key Insights
Comparison of price distribution between Convertible and Saloon vehicles
Identification of trends in automobile specifications
Insights into factors affecting vehicle pricing
Clear visualization of automobile data for better analysis

💡 Key Learnings
Data cleaning and transformation using Power Query
Creating calculated columns and measures using DAX
Designing interactive dashboards in Power BI

Presenting automobile data insights through visualization
